# 공학용 전자계산기 웹앱 PRD (Product Requirements Document)

## 1. 프로젝트 개요

### 1.1 목적
모던하고 직관적인 UI/UX를 갖춘 웹 기반 공학용 전자계산기를 개발하여, 학생, 엔지니어, 연구원 등이 복잡한 수학 및 과학 계산을 쉽고 빠르게 수행할 수 있도록 지원합니다.

### 1.2 목표
- 반응형 웹 디자인으로 다양한 디바이스에서 원활하게 작동
- 다크 모드 및 라이트 모드 지원으로 사용자 경험 향상
- 공학용 계산기의 모든 핵심 기능 제공
- 직관적이고 아름다운 UI로 사용자 만족도 극대화

### 1.3 타겟 사용자
- 대학생 및 고등학생 (수학, 물리, 공학 전공)
- 엔지니어 및 연구원
- 일반 사용자 (복잡한 계산이 필요한 경우)

## 2. 기능 요구사항

### 2.1 기본 계산 기능
- **사칙연산**: 덧셈(+), 뺄셈(-), 곱셈(×), 나눗셈(÷)
- **숫자 입력**: 0-9, 소수점(.)
- **특수 기능**:
  - `AC` (All Clear): 모든 입력 및 결과 초기화
  - `⌫` (Backspace): 마지막 입력 문자 삭제
  - `=` (Equal): 계산 실행 및 결과 표시
  - `ANS` (Answer): 이전 계산 결과 재사용
  - `EXP` (Exponent): 지수 표기법 입력

### 2.2 공학용 계산 기능

#### 삼각함수
- `sin` (사인)
- `cos` (코사인)
- `tan` (탄젠트)
- 각도 모드: `deg` (도), `rad` (라디안), `grad` (그라디안) 전환

#### 로그 및 지수
- `ln` (자연로그, base e)
- `lg` (상용로그, base 10)
- `xʸ` (거듭제곱)
- `√` (제곱근)

#### 수학 상수
- `π` (파이, 3.14159...)
- `e` (자연상수, 2.71828...)

#### 기타 함수
- `1/x` (역수)
- `%` (백분율)
- `( )` (괄호 연산)

#### 고급 기능
- `2nd` (Second Function): 역삼각함수 및 추가 함수 모드 전환
  - `sin⁻¹`, `cos⁻¹`, `tan⁻¹` (역삼각함수)
  - `x²`, `x³` (제곱, 세제곱)
  - `ⁿ√` (n제곱근)
  - `10ˣ`, `eˣ` (지수 함수)

### 2.3 디스플레이 기능
- **입력 표시**: 현재 입력 중인 수식을 상단에 표시 (예: `250 × sin(45)`)
- **결과 표시**: 계산 결과를 하단에 크게 표시 (예: `176.776`)
- **실시간 계산**: 입력과 동시에 수식 미리보기
- **긴 수식 처리**: 자동 줄바꿈 및 스크롤 지원

### 2.4 UI/UX 기능
- **다크 모드 / 라이트 모드**: 사용자 선택 또는 시스템 설정 자동 감지
- **계산 히스토리**: 이전 계산 기록 조회 및 재사용
- **메뉴**: 설정, 도움말, 정보 등 접근
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 화면 크기 지원
- **키보드 입력 지원**: 물리적 키보드로도 계산 가능
- **애니메이션 및 피드백**: 버튼 클릭 시 시각적 피드백 (scale, hover 효과)

## 3. 디자인 요구사항

### 3.1 디자인 시스템

#### 색상 팔레트
**라이트 모드**:
- 배경색: `#f6f6f8`
- 텍스트 (주요): `#0f172a` (slate-900)
- 텍스트 (보조): `#64748b` (slate-500)
- 프라이머리: `#135bec` (파란색)
- 숫자 버튼 배경: `#f1f5f9` (slate-100)
- 과학 함수 버튼 배경: `#ffffff` (흰색)
- 연산자 버튼 배경: `rgba(19, 91, 236, 0.1)` (프라이머리 10% 투명도)
- AC 버튼 배경: `#fef2f2` (red-50)
- AC 버튼 텍스트: `#ef4444` (red-500)

**다크 모드**:
- 배경색: `#101622`
- 텍스트 (주요): `#ffffff`
- 텍스트 (보조): `#94a3b8` (slate-400)
- 프라이머리: `#6fa3ff` (밝은 파란색)
- 숫자 버튼 배경: `#1e293b`
- 과학 함수 버튼 배경: `#192233`
- 과학 함수 버튼 테두리: `#324467`
- 연산자 버튼 배경: `#2d3a52`
- AC 버튼 배경: `#331e1e`
- AC 버튼 텍스트: `#f87171` (red-400)

#### 타이포그래피
- **헤더 폰트**: Space Grotesk (sans-serif)
- **본문 폰트**: Noto Sans (sans-serif)
- **아이콘**: Material Symbols Outlined

**폰트 크기**:
- 앱 제목: `18px` (font-bold)
- 입력 수식: `24px` (font-medium)
- 결과 표시: `56px` (3.5rem, font-bold)
- 숫자 버튼: `20px` (font-semibold)
- 함수 버튼: `14px` (font-bold)

#### 레이아웃
- **헤더 (Top App Bar)**:
  - 높이: 고정 (shrink-0)
  - 좌측: 메뉴 버튼 (햄버거 아이콘)
  - 중앙: 앱 제목 ("Scientific")
  - 우측: 히스토리 버튼 (시계 아이콘)

- **디스플레이 영역**:
  - 입력 수식: 우측 정렬, 회색 텍스트
  - 결과: 우측 정렬, 큰 볼드 텍스트
  - 하단 테두리로 키패드와 구분

- **키패드 영역**:
  - 5열 그리드 레이아웃 (grid-cols-5)
  - 버튼 간격: `12px` (gap-3)
  - 과학 함수 버튼 (3행): 높이 `56px` (h-14)
  - 숫자 및 연산 버튼 (4행): 높이 `64px` (h-16)

#### 버튼 스타일
- **숫자 버튼**: 라운드 코너 (`rounded-lg`), 배경색 slate-100/dark:1e293b
- **과학 함수 버튼**: 라운드 코너, 흰색 배경 + 테두리 (라이트 모드), 다크 배경 + 테두리 (다크 모드)
- **연산자 버튼**: 프라이머리 색상 배경 (투명도 10%)
- **AC 버튼**: 빨간색 배경 (투명도 낮음)
- **= 버튼**: 프라이머리 색상 배경, 그림자 효과 (`shadow-[0_4px_12px_rgba(19,91,236,0.4)]`)
- **호버 효과**: 배경색 변화 (`hover:bg-*`)
- **클릭 효과**: 스케일 축소 (`active:scale-95`)

### 3.2 반응형 디자인
- **모바일 (< 640px)**: 세로 모드 최적화, 버튼 크기 터치 친화적
- **태블릿 (640px - 1024px)**: 중간 크기 레이아웃
- **데스크톱 (> 1024px)**: 최대 너비 제한, 중앙 정렬

### 3.3 애니메이션 및 인터랙션
- 버튼 클릭 시 `scale-95` 애니메이션
- 호버 시 배경색 전환 (`transition-colors`)
- 모든 전환 효과는 부드럽게 (`transition-all`)

## 4. 기술 요구사항

### 4.1 프론트엔드 기술 스택
- **HTML5**: 시맨틱 마크업
- **CSS3**: Tailwind CSS 프레임워크 사용
- **JavaScript (ES6+)**: 계산 로직 및 UI 인터랙션
- **폰트**: Google Fonts (Space Grotesk, Noto Sans)
- **아이콘**: Material Symbols Outlined

- Safari (최신 버전)
- Edge (최신 버전)
- 모바일 브라우저 (iOS Safari, Chrome Mobile)

### 4.3 성능 요구사항

- ARIA 레이블 적용
- 색상 대비 비율 준수 (WCAG 2.1 AA 기준)
- 스크린 리더 호환성
